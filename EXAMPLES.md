## 充足率の見方

### 優秀（100-130%）🟢
- 目標値に対して適切な摂取量
- 続けましょう!

例:
```
エネルギー: 2,650kcal (目標2,700kcal) → 98%
たんぱく質: 72g (目標70g) → 103%
```

### やや不足（70-100%）🟡
- 少し意識して増やすと良い
- 目標に近づいています

例:
```
カルシウム: 620mg (目標750mg) → 83%
鉄: 6.2mg (目標7.5mg) → 83%
```

### 不足（70%未満）🔴
- 意識的に増やす必要あり
- 該当食品を多めに

例:
```
食物繊維: 14g (目標22g) → 64%
ビタミンC: 55mg (目標100mg) → 55%
```

### 過剰（130%以上）🔵
- 少し減らすことを検討
- 長期的に過剰は避ける

例:
```
食塩: 2.1g (目標1.5g) → 140%
```

## よくある質問

### Q: 食品名が見つからない場合は?
**A:** より一般的な名前で入力してください。
- 「白米」→「ご飯」
- 「豆腐」→「木綿豆腐」or「絹ごし豆腐」
- 「鶏肉」→「鶏もも肉」or「鶏むね肉」

### Q: 調理方法は考慮される?
**A:** はい、AIが適切に判断します。
- 「ゆで卵」→ 鶏卵　全卵　ゆで
- 「焼き魚」→ 該当する魚の焼きデータ
- 「生野菜」→ 該当する野菜の生データ

### Q: 複数人で使える?
**A:** はい、個人名を変えることで複数人のデータを管理できます。

### Q: データはいつまで保存される?
**A:** 無料プランでは永続化保証がありません。重要なデータは定期的にバックアップしてください。

### Q: グラム数がわからない場合は?
**A:** 一般的な目安:
- ご飯茶碗1杯: 160g
- 納豆1パック: 45g
- 卵1個: 60g
- 肉1切れ: 80-100g
- 野菜小鉢1杯: 80g
